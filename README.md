# Class Pop

🎮 **Class Pop** est un jeu de réaction rapide conçu spécialement pour les membres de ma classe. Dans ce jeu, chaque tête souriante qui apparaît à l'écran représente un membre de la classe qui a accepté d'être intégré dans le jeu. Le but est de cliquer sur ces têtes pour marquer des points. Le jeu propose des éléments bonus et malus pour ajouter du challenge et la difficulté augmente avec le score. Développé en HTML, CSS et JavaScript, ce jeu est conçu pour offrir une expérience divertissante et stimulante pour tester vos réflexes.

## Fonctionnalités

- 😁 **Gameplay Dynamique :** Les têtes souriantes apparaissent aléatoirement à l'écran et doivent être cliquées pour marquer des points.
- 🎭 **Éléments Bonus et Malus :** Des têtes bonus et malus ajoutent de la variété au jeu. Les bonus augmentent le score, tandis que les malus le diminuent.
- 🤯 **Augmentation de la Difficulté :** La fréquence d'apparition des têtes augmente à mesure que le score monte, rendant le jeu de plus en plus difficile.
- ⏱️ **Chronomètre :** Un compte à rebours limite la durée du jeu. Le jeu se termine lorsque le temps est écoulé ou que toutes les vies sont perdues.
- 👁️ **Animation et Effets sonores :** Animations pour les clics des têtes, et effets sonores pour rendre le jeu plus engageant.

## Technologies Utilisées
  
- **HTML5 :** Utilisé pour créer la structure des éléments du jeu, comme le plateau et les boutons.
- **CSS3 :**  Utilisé pour améliorer l'esthétique et l'expérience utilisateur.
- **JavaScript :**  Utilisé pour gérer les interactions de l'utilisateur, la création dynamique des têtes, les animations, et la gestion du score et du chronomètre.

## Objectifs d'Amélioration

Mon objectif pour ce jeu est de l'améliorer afin qu'il soit plus captivant et plus fluide. Les futures améliorations pourraient inclure :
- 📊 Ajouter de nouveaux niveaux avec des défis variés.
- 🎨 Intégrer des graphiques et animations plus avancés.
- 🔊 Améliorer les effets sonores et la musique de fon.
- 🏎️ Optimiser le code pour une performance encore meilleure.
- 💾 Ajouter une fonctionnalité de sauvegarde des scores pour que les joueurs puissent comparer leurs performances.
